# Sentence Enders

A collection of sentence enders that add different emphasis and implications to sentences.

| Ender     | Example | Meaning |
| --------- | ------- | ------- |
| か        | - | Marks the sentence as a question. |
| no        | - | Also a question marker. |
| ね        | - | Expects and requests agreement. |
| よ        | - | Adds an explanatory tone implying the information given is expected to be new to the recipient. |
| な        | - | Directs the sentence at oneself. |
| かい      | - | Same as `か` to form a question but this is less formal and "softer". |
| だい      | - | Same as `かい` but for `だ` ending sentences. |
| ぜ        | - | Adds strong emphasis almost like a verbal exclamation mark. |
| ぞ        | - | Adds strong emphasis almost like a verbal exclamation mark. |
| さ        | - | Similar to `ぜ` and `ぞ` but is a bit softer. |
| から      | - | Marks the sentence as a reason or explanation. |
| し        | - | Similar to `から` but is generally used to combine reasons. A single `し` implies the reason stated is not the only reason. |
| ちょうだい | - | Like `ください` but informal |
