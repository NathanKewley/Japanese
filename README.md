## Jargon Buster

| Term      | Definition                                                      |
| --------- | --------------------------------------------------------------- |
| Noun      | A Names Entity, often defined as: `A Person, Place or Thing`.   |
| Verb      | A verb describes an action, state or occurance. `A doing word`. |
| Adjective | An attribute of a noun. `A describing word`.                    |
