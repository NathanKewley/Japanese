# に

The `に` particle has multiple uses.

## に Time

The `に` particle can be used to mark what time something takes place or happens. It is important to note `に` is only used for `static` time phrases like:

* いちじに (at one o’clock)
* ごがつに (in May)
* どようびに (on Saturday)

Relative time phrases that change meaning depending on when in time they are said are **NEVER** accompanied by a particle. this is phrases like:

* きょう (today)
* らいしゅう (next week)
* せんげつ (last month)
* いちじ　から　さんじ　まで (From x until x)

Habitual phrases **DO NOT** need a particle to accompany them either:

* まいにち (everyday)
* まいとし (every year)

Eamples of `に` usage for marking time:

| Example                                  | Translation                |
| ---------------------------------------- | -------------------------- |
| じゅうじ に ねます                         | I slept at 10              |
| わたし は きんようび に てがみ を かきますた | I wrote a letter on Friday |

## に Purpose

`に` can be used to show the purpose of going somewhere. In this case it must follow a `Noun` or `Verb` which is an aciton.

The `Noun` / `Verb` is changed into its `ます` form then the `ます` is removed and replaced with `に`.

This means for words like `べんきょうします` it will have `しに` as only the `ます` is removed.

For words like `買い物` which become `買い物　を　します` they just have `に` appended.

| Example                                  | Translation                    |
| ---------------------------------------- | ------------------------------ |
| 図書館　へ　べんきょうし　に　いきます      | I came to the library to study |
| 日本　へ　買い物　に　行きます             | I go to japan to do shopping   |
