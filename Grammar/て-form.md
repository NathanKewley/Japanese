# て Form

The `て Form` links `actions`, `events`, and `states`. This is also all context dependant.

`て` is generally used to link 2 things togther.. it seems a bit ambiguas. Seems iy behaves somewhat like `and`.

`起きて トイレ に 行った` (I woke and went to the toilet).

The following all have conjugations into the `て Form`:

* Verbs
* い-Adjectives
* な-Adjectives
* Nouns

for details on how to conjugate each type please see [て Form Conjugations](て-form-conjugation.md)
