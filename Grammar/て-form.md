# て Form

Like most things in Japanese the `て Form` has multiple different uses.

## Tense for currently doing

The `て Form` can be used to say you are currently doing something.

Like `eat` --> `eating` you have `食べる` --> `食べて`

## Linking ideas

The `て Form` can be used to link `actions`, `events`, and `states`. This is also all context dependant.

`起きて トイレ に 行った` (I woke and then went to the toilet).

The following all have conjugations into the `て Form`:

## Conjugating to the て Form

* Verbs
* い-Adjectives
* な-Adjectives
* Nouns

for details on how to conjugate each type please see [て Form Conjugations](て-form-conjugation.md).
