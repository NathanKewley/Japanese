# の particle

The `の` particle as usual has multiple uses in Japanese.

## Possesion

`の` is used to show posession....

## Noun Modification

Used to allow modying a noun with a noun.

Usage: `<Noun>の<Noun>`

* `これは　日本の　ワイン` (This is Japanese wine)
* `日本語の　先生　です` ((Subject) is an Japanese teacher)

This usage is quite versatile and can include `adjectives` etc so long as a `Noun` is the main target of the modification.

Usage: `<Noun>の<Adjective><Noun>`

* `髪の 赤い 人` (Person with red hair)
