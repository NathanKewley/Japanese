# も

`も` is a non logical topic marker, this means it does not change the structure of function of the sentance. `も` only ever marks the topic of the sentance if the topic changes to say also. So a `も` this sentance applies in realtion to this new topic `as well as` the pre-existing topic.

`Aさんは　オーストラリアじん です。` (A is Australian)
`Bさんも オーストラリアじん です。` (B is also Australian)

The second sentance with `も` changes the subject of the sentance but keeps the function the same.

You can never use `も` for the `same subject` or for a `different sentance meaning`.
