# N5 Grammar

[<- BACK](./README.md)

Notes from grammar points that are expected to be in N5.

## Particles

| Particle             | Example                                                 | Summary                                               |
| -------------------- | ------------------------------------------------------- | ----------------------------------------------------- |
| は (Subject)          | わたしは Aです                                           | Marks the topic/main subject of the sentence.         |
| が (Subject)          | わたしは うまが すきです                                   | Marks a secondary subject or emphasizes the subject.  |
| を (direct Object)    | おさけを のみますか                                       | Marks the direct object of a verb.                    |
| も (Also)             | Aさんは オーストラリアじんです。Bさんも オーストラリアじんです  | Means "also" or "too". Can replace が and を directly. |
| の (Ownership)        | それは Aさんのです                                        | Shows possession or ownership.                        |
| の (Noun Substitute)  | たべものは からいのは にがてです                            | Replaces a noun already understood from context.      |
| の (Place Relativity) | いすの うえに ねこがいます                                 | Indicates location relative to another object.        |
| に (Time)             | １２じに おきました                                       | Marks a specific point in time.                       |
| に (Frequency)        | 1年に 5回ぐらい 映画をみます                               | Indicates frequency within a time period.             |
| に (Purpose)          | 渋谷へ お酒を のみに いきました                             | Indicates purpose of movement.                        |
| と (And)              | ワインと ビールを のみます                                 | Connects nouns ("and").                               |
| と (With)             | だれと 東京へ いきましたか                                 | Indicates doing something with someone.               |
| へ (Direction)        | きょうとへ 行きます                                       | Indicates direction or destination.                   |
| で (Method)           | しんかんせんで いきます                                    | Indicates means or method.                            |
| で (Location)         | えきで しんぶんを かいます                                 | Marks location where an action occurs.                |
| で (Tool)             | はしで たべました                                         | Marks the tool used for an action.                    |
| で (Group Selection)  | 日本の料理で 一番すきですか                                 | Indicates selection from a group.                    |
| で (Duration)         | にじかんで ほんを よみました                                | Indicates how long an action took to complete.       |

## Basic Grammar

| Grammar                   | Example                    | Summary                                     |
| ------------------------- | -------------------------- | ------------------------------------------- |
| です                      | わたしは Aです               | Polite statement ending.                    |
| ですか                    | あのひとは だれですか          | Polite question ending.                     |
| よ                        | あまり あつくないですよ        | Indicates new information for the listener. |
| じゃありません              | わたしは がくせいじゃありません | Polite negative of です.                    |
| あります                   | りんごが あります             | Existence of inanimate objects.            |
| います                     | ねこが います                | Existence of animate objects.              |
| さん                       | Aさん                       | Polite honorific.                          |
| これ・それ・あれ             | これは ほんです              | Demonstrative pronouns.                    |
| この・その・あの             | このかばんは わたしのです     | Demonstrative adjectives.                   |
| だれ                       | これは だれのですか          | "Who".                                      |
| なん                       | これは なんですか            | "What".                                     |
| なんの                     | それは なんのざっしですか     | "What kind of".                             |
| ちがいます                  | いいえ ちがいます            | "That is incorrect".                       |
| ここ                       | ここは がっこうです          | Here.                                      |
| どこ                       | がっこうは どこですか        | Where.                                      |
| こちら・そちら・あちら・どちら | がっこうは あちらです        | Polite directional forms.                  |
| から～まで                  | １２じから １じまでです      | From ~ to ~.                               |
| 時・分                     | １２時                     | Time expressions.                          |
| Xも                       | なにも たべません            | "None / not any".                          |
| いつ                      | いつ にほんへ いきましたか    | When.                                       |
| いっしょに                 | いっしょに のみませんか       | Together.                                   |
| ～ませんか                 | のみませんか                | Invitation form.                            |
| ～ましょう                 | のみましょう                | Let's do.                                   |
| それから                   | それから えいがをみました     | After that.                                |
| もう                      | もう たべましたか            | Already.                                    |
| とくい                     | うたが とくいです           | Good at.                                    |
| にがて                     | うたが にがてです           | Bad at / dislike.                           |
| どんな                     | どんな たべものが すきですか | What kind of.                                |
| どう                       | べんきょうは どうですか     | How is it?                                   |
| こと                       | 泳ぐこと                  | Nominalizes a verb.                          |
| ことができます              | 空手ができます              | Can do.                                     |
| ことがあります              | 飛行機に乗ったことがあります  | Have experienced.                           |
| ～たことがありません         | 食べたことがありません       | Have never done.                            |

## Giving and Recieving

| Grammar | Example         | Summary                |
| ------- | --------------- | ---------------------- |
| あげます    | はなを あげました       | To give.               |
| もらいます   | ほんを もらいました      | To receive.            |
| おくります   | にもつを おくりました     | To send.               |
| かります    | ほんを かります        | To borrow.             |
| かします    | ほんを かします        | To lend.               |
| おしえます   | Aさんは Bさんに おしえます | To teach.              |
| ならいます   | テニスを ならいました     | To learn from someone. |

## Connecting Ideas

| Grammar | Example            | Summary                     |
| ------- | ------------------ | --------------------------- |
| そして     | きれいです。そして ひろいです。   | Connects similar ideas.     |
| けど      | おもしろいですけど むずかしいです  | Connects contrasting ideas. |
| から      | いそがしいですから テレビをみません | Because.                    |
| どうして    | どうして いきませんか        | Why / How.                  |

## Comparisons

| どちらが    | 犬と猫とどちらがすきですか  | Which of two. |
| どちらも    | どちらも すきです      | Both.         |
