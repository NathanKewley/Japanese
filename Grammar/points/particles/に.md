# に - Particle

The `に` particle has multiple uses:

* Time
* Target of an Action
* Target of a Change
* Purpose

## に Time

The `に` particle can be used to mark what time something takes place or happens. It is important to note `に` is only used for `static` time phrases like:

* いちじに (at one o’clock)
* ごがつに (in May)
* どようびに (on Saturday)

Relative time phrases that change meaning depending on when in time they are said are **NEVER** accompanied by a particle. this is phrases like:

* きょう (today)
* らいしゅう (next week)
* せんげつ (last month)
* いちじ　から　さんじ　まで (From x until x)

Habitual phrases **DO NOT** need a particle to accompany them either:

* まいにち (everyday)
* まいとし (every year)

Eamples of `に` usage for marking time:

| Example                                  | Translation                |
| ---------------------------------------- | -------------------------- |
| じゅうじ に ねます                         | (I) slept at 10            |
| わたし は きんようび に てがみ を かきますた | I wrote a letter on Friday |

## に Target of an Action

`に` can be used to mark the target of an action. Sending something somewhere, putting something somewhere, throwing something somewhere, etc. Any time something is going somewhere. Some examples of this:

* `学校　に　いく` ((I) will go to school)
* `さくらさん　に　ボール　を　なげた` ((I) Threw the ball at Sakura)

## に Target of a Change

`に` can also mark the target of a change or transformation.

* `さくらさん　わ　かえる　に　なった` (Sakura become a Frog) <sup>1</sup>

In this example the Frog is marked with `に` as that is what Sakura was turned into, the target of the change was to become a Frog.

* `ことし　十八歳　に　なる` (This year (I) become 18) <sup>1</sup>

<sub>1. [なった is past tenst of なる which means to become.]</sub>

## に Purpose (Which is still a Target in a way)

`に` can be used to show the purpose of going somewhere. In this case it must follow a `Noun` or `Verb` which is an aciton.

```text
This feels like just for super formal stuff and can prob mostly be disregarded?

The `Noun` / `Verb` is changed into its `ます` form then the `ます` is removed and replaced with `に`.

This means for words like `べんきょうします` it will have `しに` as only the `ます` is removed.

For words like `買い物` which become `買い物　を　します` they just have `に` appended.
```

> [!NOTE]
> へ in these examples is replaceable with に.
> へ as a particle is only ever usable in this way so feels a bit redundant.

| Example                                  | Translation                     |
| ---------------------------------------- | ------------------------------- |
| 図書館　へ　べんきょうし　に　いきます      | (I) came to the library to study |
| 日本　へ　買い物　に　行きます             | (I) go to japan to do shopping   |
| おみせ　に　たまご　を　かいに　いく        | (I) go to the shops to buy eggs  |
