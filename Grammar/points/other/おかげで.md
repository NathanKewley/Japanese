# ～おかげで - Because Of (Positive Cause)

[<- Back](../../README.md)

`～おかげで` indicates that **Thanks to A, good thing B happened**.

## Forms and Usage

| Type         | Rule                     | Example        |
| ------------ | ------------------------ | -------------- |
| Noun         | Noun + のおかげで          | 先生のおかげで   |
| い-Adjective | い-Adjective + おかげで    | 忙しいおかげで   |
| な-Adjective | な-Adjective + な おかげで | 元気なおかげで   |
| Verb         | Plain Form + おかげで     | 勉強したおかげで |

## Examples

先生のおかげで日本語が上手になった。  
Thanks to my teacher, I became good at Japanese.

毎日勉強したおかげで試験に合格した。  
Thanks to studying every day, I passed the exam.

天気が良かったおかげで楽しい旅行になった。  
Thanks to the good weather, it turned into an enjoyable trip.

友達が手伝ってくれたおかげで早く終わった。  
Thanks to my friend helping me, I finished early.

健康なおかげで毎日元気に働ける。  
Because I'm healthy, I can work energetically every day.

## Notes

- `～おかげで` is used when the **result is positive or desirable**.
- For a **neutral** cause, use **～ので** or **～から** instead.
- For a **negative** cause, use **～せいで** instead.
