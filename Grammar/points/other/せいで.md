# ～せいで - Because Of (Negative Cause)

[<- Back](../../README.md)

`～せいで` indicates that **Because of A, Bad thing B Happend**, often implying blame, frustration, or regret.

## Forms and Usage

| Type        | Rule                   | Example     |
| ----------- | ---------------------- | ----------- |
| Noun        | Noun + のせいで         | 雨のせいで    |
| い-Adjective | い-Adjective + せいで   | 暑いせいで   |
| な-Adjective | な-Adjective + なせいで | 静かなせいで |
| Verb         | Plain Form + せいで    | 遅れたせいで |

## Examples

雨のせいで試合が中止になった
Because of the rain, the game was cancelled.

寝坊したせいで電車に乗り遅れた
Because I overslept, I missed the train.

彼のミスのせいでみんなが困った
Because of his mistake, everyone had trouble.

暑いせいで全然眠れなかった
Because it was so hot, I couldn't sleep at all.

この道は狭いせいで、よく渋滞する
Because this road is narrow, it often gets congested.

## Notes

- `～せいで` is used when the **result is negative or undesirable**.
- For a **neutral** cause, use **～ので** or **～から** instead.
- For a **positive** cause use **～おかげで** ("thanks to")
