# ～たがる / ～たがった - “Shows someone else’s desire”

[<- Back](../../README.md)

`～たがる` is used to express that **someone else seems to want to do something**, based on observation. It is not used for the speaker’s own desires.

`～たがった` is the **past form**, meaning someone **seemed to want to do something**.

## Forms and Usage

`～たがる` is attached to the **verb stem**.

| Form           | Structure   | Meaning                           |
| -------------- | ----------- | --------------------------------- |
| Present        | ～たがる     | seems to want to…                 |
| Past           | ～たがった    | seemed to want to…               |
| Polite Present | ～たがります   | seems to want to… (polite)       |
| Polite Past    | ～たがりました | seemed to want to… (polite past) |

## Formation

| Verb Type  | Rule                           | Example              |
| ---------- | ------------------------------ | -------------------- |
| Ichidan    | Remove る + たがる               | 食べる → 食べたがる    |
| Godan      | ます stem + たがる               | 行く → 行きたがる      |
| Irregular  | する → したがる<br>来る → 来たがる | 勉強する → 勉強したがる |

## Examples

彼は日本に行きたがっている。  
He seems like he wants to go to Japan.

子どもはお菓子を食べたがる。  
Children tend to want to eat sweets.

彼女は写真を撮りたがった。  
She seemed like she wanted to take photos.

弟はゲームを買いたがっている。  
My younger brother seems to want to buy a game.
