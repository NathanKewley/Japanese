# か

`か` had multiple uses as a particle.

## か - Question Marker

`か` can be appended to the end of a statement to make it a question

## か - Choice Marker

`か` can be used to mark the choice between a pair of options. like `or` in english. It can be used in the following formulas:

### (noun)か(noun)

クッキー `か` ケーキ を たべたい です (I want to either a cookie or cake)

### (adjective)か(adjective)

あつい `か` さむい ひ が すき です か (Do you like hot or cold days)

### (verb)か(verb)か

パーティー に いく `か` いえ に かえる `か` わかりません (idk if I should go to the party or just go home)

Note: when using `か` with `(verb)か(verb)か` it is required to put `か` at the end of both verbs.
