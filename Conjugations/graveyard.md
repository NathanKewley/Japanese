## Other Forms

First row in the table is the plan dicitonary for of an example Ichidan and Godan verb.

| Form                 | Purpose        | Ichidan      | Godan      |
| -------------------- | -------------- | ------------ | ---------- |
| Dictionary (example) | Plain          | 食べる        | 話す        |
| ます                 | Polite         | 食べます       | 話します    |
| ない                 | Negative       | 食べない       | 話さない    |
| たい                 | Want To        | 食べたい       | 話したい    |
| そう                 | Looks Like     | 食べそう       | 話しそう    |
| やすい               | Easy To        | 食べやすい     | 話しやすい   |
| にくい               | Difficult To   | 食べにくい     | 話しにくい   |
| なさい               | Polite Command | 食べなさい     | 話しなさい   |
| Potential           | Can Do         | 食べられる     | 話せる       |
| Passive             | Be Done To     | 食べられる     | 話される     |
| Causative           | Make/Let Do    | 食べさせる     | 話させる     |
| Causative Passive   | Be Made To Do  | 食べさせられる  | 話させられる |
| Conditional         | If             | 食べれば       | 話せば      |
| Volitional          | Let's          | 食べよう       | 話そう      |
| Imperative          | Command        | 食べろ / 食べよ | 話せ       |
| Prohibition         | Don't Do       | 食べるな       | 話すな      |
| たり                 | Do Things Like | 食べたり       | 話したり    |
